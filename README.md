# 【擊殺鬼王】大富翁

> 2026 新生教會主日學製作

專為大螢幕投影設計的夏令營互動大富翁遊戲。6 隊玩家化身鬼滅之刃六大柱，合力將鬼王無慘的血量打至歸零。

🎮 **線上遊玩**：https://sallysywang.github.io/MONOPOLy/

---

## 遊戲簡介

這是一場全員協作的戰役。鬼王代表讓心無法原諒的黑暗力量，玩家必須透過完成各種任務對鬼王造成傷害。鬼王血量歸零後，全員齊喊「鬼殺隊——斬！」發動最終一擊。

- **人數**：6 隊 + 1 名工作人員操作電腦
- **設備**：投影機、筆電、實體骰子（D6）× 2
- **勝利條件**：合力將鬼王 100 HP 打至歸零

## 遊戲規則

### 棋盤
16 格環狀棋盤，分為三種格子：
- 🏁 **START**（格 1）：起始點，可執行任務
- ⚔️ **任務格**：完成任務扣鬼王 HP（5 或 10 點）
- 🎴 **卡牌格**（格 10、12、16）：翻牌抽機會卡或命運卡

### 回合流程
每大輪由 6 隊依序行動，最後輪到鬼王：

**玩家回合：**
1. **（若有待結算任務）** 先公告上回合任務結果 → 工作人員點「✅ 任務成功」或「❌ 任務失敗」
2. 擲實體骰子（D6），工作人員輸入點數 → 棋子移動
3. 依格子類型行動：
   - **任務格**：公告任務內容 → 點「🏃 出發執行任務！」→ 換下一位（**結果留待下一輪揭曉**）
   - **卡牌格**：翻牌抽卡，效果立即生效

**鬼王回合：**
1. 擲骰移動鬼王棋子
2. 系統自動計算距離最近的兩隊為攻擊目標
3. 再擲一次骰子決定傷害（1~6 點 × 倍率）

### 卡牌系統
停在卡牌格時，翻開 4 張牌中的 1 張（50% 機會卡 / 50% 命運卡）：

**機會卡（對玩家有利）**
| 卡名 | 效果 |
|------|------|
| 死者甦醒 | 復活一隊陣亡小組，HP 重置為 5 |
| 生命藥水 | 本組回復 5 HP（上限 10）|
| 防護盾 | 下一次鬼王攻擊對本組無效 |
| 強化攻擊 | 本行動任務成功時傷害加倍 |

**命運卡（對鬼王有利）**
| 卡名 | 效果 |
|------|------|
| 夜晚來臨 | 鬼王下一回合攻擊傷害 ×2 |
| 白天來了 | 鬼王下一回合暫停行動 |
| 鬼放大招 | 鬼王對所有存活隊伍造成 3 點傷害 |
| 真是個平安的一天 | 無事發生 |

## 操作說明（工作人員）

| 操作 | 說明 |
|------|------|
| 結算上回合任務 | 輪到某隊時，若上回合有任務，先點「✅ 任務成功」或「❌ 任務失敗」|
| 擲骰 | 點擊 1~6 對應數字按鈕 |
| 出發執行任務 | 落在任務格後點「🏃 出發執行任務！」，結果留待下一輪揭曉 |
| 翻牌 | 落在卡牌格後點「🎴 翻牌」，選一張點擊翻開，效果立即生效 |
| 換下一位 | 點「換下一位 →」進入下一隊 |
| ↩ 上一步 | 操作失誤時可還原上一個動作 |
| 💾 存檔/讀檔 | 產生 JSON 存檔，或貼上 JSON 還原進度 |

## 終局

**玩家勝利：** 鬼王 HP 歸零 → 全螢幕顯示「鬼殺隊——斬！」儀式畫面 → 傷害貢獻排行榜，最高傷害隊伍獲得「最強鬼殺隊員」稱號。

**鬼王勝利：** 所有隊伍 HP 同時歸零。

## 技術規格

- 純前端單一 HTML 檔案（無後端、無資料庫）
- Vue.js 3 + Tailwind CSS
- 16:9 比例鎖定，適合投影機顯示
- 暗黑風格 UI，高對比配色確保後排可見
- 支援 JSON 存讀檔（中途關閉瀏覽器可還原）
