# 【擊殺鬼王】大富翁

**2026 堅持夏趣 · 無限城篇**

新生教會主日學 2026 年夏趣活動專用的鬼滅之刃主題大富翁遊戲。6 隊玩家化身六大柱，輪流行動、完成任務，合力將鬼王無慘的 HP 打至歸零，最後全員齊喊「鬼殺隊──斬！」

---

## 如何開始玩

直接用瀏覽器開啟 `index.html`，無需安裝任何套件。

```
open index.html
```

---

## 遊戲特色

- **6 隊對戰**：霞・水・炎・蛇・蟲・風六柱，各有專屬代表色
- **鬼王 AI**：自動計算最近兩隊發動攻擊，支援傷害倍率狀態
- **機會 / 命運卡**：死者甦醒、防護盾、強化攻擊等 8 張卡牌系統
- **任務延遲結算**：落在任務格出發，下一輪才揭曉成敗，還原現場臨場感
- **自動儲存**：遊戲狀態存入 localStorage，重開頁面可繼續
- **存讀檔**：支援 JSON 格式手動存檔與還原
- **上一步**：操作失誤可還原至上一動作
- **逐格移動動畫**：棋子逐格前進動畫
- **鎹烏送信動畫**：任務出發時觸發鎹烏飛行動畫
